# Markpane Homebrew Tap

Homebrew tap for the [Markpane](https://markpane.com) CLI.

```sh
brew tap seba-rb/markpane
brew install markpane
```

`markpane` gives terminal agents and scripts access to your Markpane
workspaces: list and evaluate smart views, run frontmatter queries, render
Agent Briefs, and create views that appear in the app sidebar.

Requires macOS 14+ and the Markpane app (register your workspaces there
first).
